# BOJ 골드4 1987 알파벳

### 풀이 요약

- 풀이 시간 : 측정 x
- 풀이 방법 : DFS
- 메모리 : 16232KB, 시간 : 1124ms

<br>

### 풀이 방법

좌측 상단에서 시작한 말이 최대한 몇 칸을 지날 수 있는지 구하는 프로그램이다. 보드의 각 칸에는 대문자 알파벳이 적혀 있고, 지금까지 지나온 모든 칸에 적혀있는 알파벳과 같은 알파벳을 지날 수 없다. 

<br>

1. R과 C 입력
2. 알파벳 보드인 map 을 입력한다. 
3. 방문 처리를 위한 배열 visited와 alphabet 방문 처리 배열인 alphabet을 생성한다. 
4. 좌측 상단 0,0 부터 시작하여 DFS를 통해 최대한 몇 칸을 지날 수 있는 지 구한다. 
    1. 현재의 칸에서 상 하 좌 우 인접한 네 칸으로 이동하도록 반복문을 진행한다. 
    2. 이동할 수 있는 인접한 칸은, 범위 안에 있으며 방문한 적 없고, 이전에 없던 알파벳이어야한다. 
    3. 방문 처리를 하고 다음으로 이동한다. 
    4. 한 칸에서 다른 칸으로 또 이동할 수 있으므로 visited 배열과 알파벳 배열을 false 처리해준다.
5. DFS 탐색이 끝났으면 구한 최대 이동 수를 출력한다.
