# BOJ 실버1 15903 카드 합체 놀이

### 풀이 요약

- 풀이 시간 : 약 15분
- 풀이 방법 : 그리디
- 메모리 : 31884KB, 시간 : 576ms

<br>

### 풀이 방법

1. n, m 입력 받기 
2. 카드에 적힌 수 입력 받기 
3. 카드 합체 과정
    1. 카드를 합체해서 나온 값 중 제일 작은 값을 만드려면 카드를 합체할 때, 제일 작은 수 두개를 골라서 더해줘야한다. 
    2. 카드를 정렬한다.
    3. 정렬한 후 인덱스 0과 1의 값을 더한 후, 더한 값을 arr[0]과 arr[1]에 넣어준다.
4. 카드 놀이가 끝난 후, 카드의 총 합을 구한다.
5. 출력한다.
