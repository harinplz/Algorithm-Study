- 풀이 시간 : 측정 X
- 풀이 방법 : Set 자료구조

### 풀이 방법

1. A 집합 크기 n과 B 집합 크기 m을 입력받는다.
2. Set을 생성한다. 조건에 숫자를 증가하는 순서대로 출력하라는 것이 있었으므로, TreeSet으로 선언한다.
3. A 집합을 입력받는다.
4. B 집합 요소를 입력받는데, 만약 set에 포함되어 있는 수라면 제거한다. 
5. set의 size와 set의 요소를 출력한다.
