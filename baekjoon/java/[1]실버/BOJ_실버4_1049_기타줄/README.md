- 메모리 : 11592KB, 시간 : 76ms

### 풀이 방법

1. N,M 을 입력받는다.
2. 브랜드의 세트와 낱개의 가격을 입력받으면서, 각각의 최소 가격을 갱신한다.
3. for문을 통해 세트를 0개 샀을 때 부터 N/6 + 1 개 샀을 때 까지의 가격을 책정하며 최소 가격을 갱신한다.
4. 최소 가격을 출력한다.
